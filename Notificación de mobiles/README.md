MobilePayAlerts

----

**Requirements:**  
- python 3.6+
- PySimpleGUIQt (requires PyQt5)
- websockets V6.0.0 (V7.0.0 doesn't work when receiving Pushbullet data)
- webbrowser
- requests
- asyncio
- logging
- qrcode
- json

`pip install PySimpleGUIQt webbrowser asyncio logging qrcode json websockets==6.0.0`
